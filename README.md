<h1>Projeto Login</h1>
<p>Este projeto, criado em conjunto ao Curso em Vídeo para um desafio do Curso de HTML5 e CSS3, foca em apresentar uma página de login priorizando o aperfeiçoamento em Media Queries.</p>

<h2>Sobre o site</h2>
<p>O site mostrará ao usuário uma página, como as de plataformas online, para a realização de um login comum. o projeto disponibilizará campos de Email e Senha, além de um parágrafo com o "Esqueci a senha". Não haverá direcionamento, o foco é apenas no loyalt e no redimensionamento quanto a troca de telas, conhecido como Media Query.</p>
